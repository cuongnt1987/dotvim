# How to install

**Installation:**

```bash
git clone git@github.com:cuongnt1987/dotvim.git ~/.vim
```

**Create symlinks:**
```bash
ln -s ~/.vim/.vimrc ~/.vimrc
ln -s ~/.vim/.vimrc ~/.gvimrc
```

**Switch to '~/.vim/' directory and fetch submodules:**

```bash
cd ~/.vim
git submodule init
git submodule update
```
